Ibex

A typed DSL for fast table pipelines.

Ibex is a statically typed language for columnar DataFrame and TimeFrame work. It uses data.table-inspired brackets and transpiles to C++23 for performance.

What Ibex focuses on

  • Named clauses like filter, select, update, by, window.
  • Predictable semantics and static typing.
  • Time-aware operations with TimeFrame and rolling windows.
  • Interop through extern fn hooks to C++.

Current language shape

Ibex keeps SQL keywords out of its surface syntax and avoids macro magic. The core unit is an expression pipeline that lowers cleanly into IR operators like scan, filter, project, aggregate, update, window, and join.

Sample Ibex

let tf = as_timeframe(ticks, "ts");

let bars = tf[resample 1m, select {
  close_1m = last(price)
}];

tf asof join bars on ts;

The as-of join keeps each left row and attaches the latest matching right row at or before that timestamp.
